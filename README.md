Detonate your Fire Crackers with Mobile Phone

1. 2000mha 3.7V battery
2. 600 mha 5v battery 
3. jumper wire 
4. coil 0.3 ~ 0.5 copper wire 
5. DC to AC Relay module
6. hc-05 Bluetooth Module 
7. arduino pro mini, UNO or arduino nano
